# Operator Groups

*Definition*: Let $G$ be a group, $\Omega$ be a set.

- **$\Omega$-endomorphism**: a map $\alpha: $

- **Left operator group**: a triple $(G,\Omega,\alpha)$ in which

    1. $G$ is a group
    2. $\Omega$ is a set (called the **operator domain**)
    3. $\alpha: \Omega \times G \to G$ is an *endomorphism* of $G$ for each fixed $\omega \in \Omega$
